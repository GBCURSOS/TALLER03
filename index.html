Ejercicio 1: Diseño con párrafos estilizados y bordes

Crear una página web que contenga tres párrafos de texto, cada uno con 50 palabras. Aplicar las siguientes características:

  1. El primer párrafo debe tener fondo amarillo, texto azul, ancho de 300px, alto automático, padding de 40px, margen superior e inferior de 10px, margen izquierdo y derecho de 20px, y un borde visible de color negro.
  2. El segundo párrafo debe tener fondo verde claro, texto rojo oscuro, ancho de 400px, padding de 10px arriba y abajo, centrado horizontalmente, y un borde visible de color gris.
  3. El tercer párrafo debe tener fondo azul claro, texto negro, ancho del 50% del contenedor, padding de 20px, centrado horizontalmente, y un borde visible de color marrón.

Ejercicio 2: Imágenes con display block y bordes

Crear una página web que incluya seis imágenes de https://picsum.photos , todas de tamaño 200x200 píxeles. Las tres primeras imágenes deben mostrarse utilizando display: block. Las tres últimas se mostrarán con su valor por defecto. Cada imagen debe tener un borde visible que permita distinguirla claramente del fondo.

Ejercicio 3: Elementos span en columna con display y bordes

Crear una página web que contenga tres elementos <span> dentro de un contenedor. Estos elementos deben aparecer uno debajo del otro, como si fueran bloques, aunque sean elementos en línea por naturaleza. Cada elemento debe tener estilo básico con color de fondo, padding y un borde visible que permita diferenciarlos visualmente. Utilizar únicamente CSS para lograr el diseño solicitado.

Ejercicio 4: Comparación entre margin y padding con bordes

Crear una página web que muestre dos elementos <div> con contenido de texto similar. Cada elemento debe tener dimensiones definidas, el mismo estilo base, y cumplir las siguientes condiciones:

  • Al primer elemento se le aplica únicamente margen superior e inferior.
  • Al segundo se le aplica únicamente padding.
  • Ambos elementos deben tener un borde visible que facilite la observación del espacio afectado por cada propiedad.
  • El diseño debe permitir comparar visualmente cómo afecta cada propiedad al espacio dentro y alrededor de los elementos.

Ejercicio 6: Colores

Modelo Descripción
RGB Es un modelo de color basado en la combinación de tres colores primarios: rojo (Red), verde (Green) y azul (Blue). Cada canal tiene valores entre 0 y 255.
HSL Representa el color en términos de matiz (Hue), saturación (Saturation) y luminosidad (Lightness). Es una forma más intuitiva de manipular el color, especialmente útil para ajustar tonos en diseño web o gráfico.
CMYK Es un modelo sustractivo utilizado en impresión. Sus componentes son cian (Cyan), magenta (Magenta), amarillo (Yellow) y negro (Key). Se usa para producir colores reales en medios impresos.
Código de Color Hexadecimal Es un código hexadecimal que representa un color en formato RGB. En este caso, #FFFFFF corresponde al color blanco, donde cada componente (rojo, verde y azul) está al máximo valor (FF en hexadecimal es 255 en decimal).

Paleta de 4 colores + texto = 5 elementos, pero solo 4 colores base:

  1. Dominante (60% aprox.). Se usa en elementos principales como encabezados, navegación y secciones destacadas.
  2. Secundario (30% aprox.). Se usa en botones, tarjetas, menús laterales u otros elementos importantes pero menos protagonistas que el color dominante.
  3. Acento (10% aprox.). Se usa en elementos clave como llamados a la acción, iconos o mensajes importantes. Su uso debe ser moderado para no saturar la interfaz.
  4. Neutro (fondo o gris base). Es el color base del sitio, usado mayormente en fondos y texto. Ayuda a equilibrar visualmente los demás colores y mejora lalegibilidad.
  5. El color del texto puede ser: Un valor derivado del neutro (por ejemplo, un gris oscuro) O una versión modificada de uno de los colores base (por ejemplo, blanco sobre fondo oscuro) Por eso, aunque se usen cinco colores (dominante, secundario, acento, fondo y texto), solo se necesitan cuatro colores base . El color del texto puede ser una variación (tono, saturación o brillo) de uno de ellos, por ejemplo.
